# Responsive Landing Page with Interactive Navigation

A simple and modern **responsive landing page** built using **HTML, CSS, and JavaScript**, featuring a fixed navigation bar with hover effects and style changes on scroll.

## Features
 Fixed and interactive navigation bar
 Scroll-based navbar color change
 Hover effects on menu items
 Fully responsive design
 Clean and modern UI

## Technologies Used
 HTML5
 CSS3
 JavaScript

## Project Structure

responsive-landing-page
├── index.html
├── style.css
└── script.js


## How to Run
1. Download or clone the repository  
2. Open `index.html` in any web browser  

## Purpose
This project demonstrates fundamental **front-end web development skills**, responsive design, and basic JavaScript interactivity.
